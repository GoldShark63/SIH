# SIH 2025 – Real Time Traffic Monitoring

## Project Description
This project is developed under Smart India Hackathon (SIH) 2025.

It focuses on real-time traffic monitoring using intelligent data processing
to analyze traffic flow and congestion.

## Folder Structure
- backend/ – Server-side logic
- instance/ – Configuration and runtime data

## How to Run
1. Navigate to backend folder
2. Install dependencies
3. Run the server
